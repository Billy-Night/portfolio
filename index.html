Weather Wise

Weather Wise is a weather & sport application, that will provide the user with a rating on how good the conditions are for that day in their selected sport, which is calculated using an algorithm, which depends on certain weather condition in their current selected location. The application uses API's to get current weather and location information. Click to see a demo of the Web App, click here to see the GitHub repository.

Client Project

TROWIT

The TROWIT project has been created to improve the transfer of business and personal details between people. It fills this need by passing user information digitally in the form of Business Cards. The user can set-up an account and create their digital business cards, they can then order these cards as physical cards, which is a service TROWIT will provide. The user can connect with other users via their cards with a URL link. Apologies but the repository is private, I would be happy to discuss the project on request.
