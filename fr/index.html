World Surf Journal

Le World Surf Journal est une application web qui permet à l'utilisateur de créer un compte et d'enregistrer ses voyages de surf avec divers détails. C'est une application CRUD, et l'utilisateur peut créer, remplacer, mettre à jour et supprimer ses voyages de surf. Il s'agit d'une application fullstack composée d'un frontend, d'un backend et d'une base de données. Découvrez la déployée web app. Le dépôt du frontend est disponible sur GitHub et le backend repository.

Weather Wise

Weather Wise est une application météo et sport qui fournit à l'utilisateur une évaluation de la qualité des conditions météo pour sa pratique sportive choisie, calculée à l'aide d'un algorithme basé sur certaines conditions météorologiques de sa localisation actuelle. L'application utilise des API pour obtenir les informations météo et de localisation actuelles. Live: web app, Le dépôt GitHub du frontend est disponible ici: GitHub repository.

Projet de client

TROWIT

Le projet TROWIT a été créé pour améliorer le transfert d'informations professionnelles et personnelles entre les personnes. Il répond à ce besoin en transmettant les informations des utilisateurs sous forme de cartes de visite numériques. L'utilisateur peut créer un compte et concevoir ses cartes de visite numériques, puis les commander sous forme de cartes physiques, un service que TROWIT proposera. L'utilisateur peut se connecter avec d'autres utilisateurs via leurs cartes grâce à un lien URL. Désolé, mais le dépôt est privé. Je serai heureux de discuter du projet sur demande.
