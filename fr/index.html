World Surf Journal

Le World Surf Journal est une application web qui permet à l'utilisateur de créer un compte et d'enregistrer ses voyages de surf avec divers détails. C'est une application CRUD, et l'utilisateur peut créer, remplacer, mettre à jour et supprimer ses voyages de surf. Il s'agit d'une application fullstack composée d'un frontend, d'un backend et d'une base de données. Découvrez la web app déployée. Le dépôt du frontend est disponible sur Github repository et le backend Github repository.

Weather Wise

Weather Wise est une application météo et sport qui fournit à l'utilisateur une évaluation de la qualité des conditions météo pour sa pratique sportive choisie, calculée à l'aide d'un algorithme basé sur certaines conditions météorologiques et de sa localisation actuelle. L'application utilise des API pour obtenir les informations météo et de localisation actuelles. Live: web app, Le dépôt GitHub du frontend est disponible ici: GitHub repository.
